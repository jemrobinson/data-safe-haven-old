This is a list of all the R packages that will be installed in the compute VM. It includes packages from various sources, including the standard library, this quick list of useful packages, packages used in previous DSGs, rstan, and finally a list of requirements from an electronic health records project. In addition, Bioconductor will be installed, together with the packages listed below, which are taken from the 50 most widely used packages. NB. Bioconductor has a 6-month release schedule so packages may need to be updated accordingly.
