Converting documentation to PDF¶

There are several ways to make shareable PDF files from the documents above. The easiest way to make shareable PDF files from the Markdown documents included here is using the markdown2pdf.sh script.

  • npm method [recommended]

    • Install npm

    • Install pretty-markdown-pdf with npm install pretty-markdown-pdf with the -g flag if you want it installed globally

    • Run ./markdown2pdf.sh <file name>.md npm

  • LaTeX method

    • Install XeLaTex, generally by installing a full LaTeX environment such as TexLive (Windows / Linux) or MacTex (macOS).

    • Install Pandoc`

    • Install the Symbola font (https://dn-works.com/ufas/)

    • Run ./markdown2pdf.sh <file name>.md latex
