NHS DSPT Security Assessment¶

This is the NHS DSPT self-assessment for the Alan Turing Institute’s deployment of a Data Safe Haven. Note that that the first four columns are generic to any Data Safe Haven deployment, while the final column is organisation-specific.

Status Key

  • Y: Yes

  • N: No

  • ?: Don’t Know

  • P: Partly

  • NA: Not Applicable

Likely Solution Key:

  • PD: Policy Document

  • TD: Technical Documentation

  • TS: Technical Solution
